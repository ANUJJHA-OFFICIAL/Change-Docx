## 🚀 Features

- 🔐 Firebase Authentication (Email/Password + Google Sign-In)
- 📧 Email Verification on Register
- 🌙 Dark Mode / Light Mode
- 📄 15+ PDF Tools — all run in your browser
- 📁 Local file upload only — files never leave your device

---

## 🛠️ PDF Tools Available

| Tool | Description |
|---|---|
| Merge PDF | Combine multiple PDFs into one |
| Split PDF | Extract pages or ranges |
| Compress PDF | Reduce file size |
| PDF to Word | Convert PDF to editable DOCX |
| Word to PDF | Convert DOCX to PDF |
| PDF to JPG | Convert pages to images |
| JPG to PDF | Convert images to PDF |
| Rotate PDF | Rotate pages 90/180/270° |
| Add Watermark | Add text watermark to all pages |
| Add Page Numbers | Number pages automatically |
| Protect PDF | Lock PDF with a password |
| Unlock PDF | Remove PDF password |
| OCR PDF | Make scanned PDF searchable |
| Excel to PDF | Convert spreadsheet to PDF |
| PDF to Excel | Extract PDF tables to XLSX |

## 🔧 Run Locally

**Prerequisites:** Node.js

1. Clone the repo:
```bash
   git clone https://github.com/ANUJJHA-OFFICIAL/Change-Docx.git
   cd Change-Docx
```

2. Install dependencies:
```bash
   npm install
```

3. Add your Firebase config in `js/firebase-config.js`

4. Run the app:
```bash
   npm run dev
```

---

## 🚢 Deploy
```bash
npm run deploy
```

Deploys to GitHub Pages automatically.
